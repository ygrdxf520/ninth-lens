# 万相 2.7 图像（wan2.7-image / wan2.7-image-pro)

万相 2.7 图像模型,支持**文生图、文生组图、图生组图、图像编辑、多图参考生成、交互式编辑(bbox)**。

通用 API 模式详见 [API 概览.md](./API%20概览.md)。

## 模型对照表

| 模型 | 简介 | 最高分辨率 |
|------|------|------------|
| `wan2.7-image-pro` | 专业版,文生图(非组图)支持 4K 高清输出 | 4K |
| `wan2.7-image` | 加速版,生成速度更快 | 2K |

**输出格式**:PNG。

**核心功能**:
- 文生图(`messages.content` 仅 `text`,无 `image`)
- 图像编辑(`messages.content` 有 `image` + `text`)
- 交互式编辑(在编辑基础上传 `bbox_list` 框选区域)
- 组图生成(`enable_sequential=true`)
- 多图参考(传多张 `image`,模型综合参考)

## 同步接口

### Endpoint

```
POST /api/v1/services/aigc/multimodal-generation/generation
```

```json
    "size": "2K",
    "n": 1,
    "watermark": false,
    "thinking_mode": true
  }
}
```

### 请求体(组图生成)

```json
{
  "model": "wan2.7-image-pro",
  "input": {
    "messages": [
      {
        "role": "user",
        "content": [
          {"text": "电影感组图,记录同一只流浪橘猫... 第一张:春天... 第二张:夏天..."}
        ]
      }
    ]
  },
  "parameters": {
    "enable_sequential": true,
    "n": 4,
    "size": "2K"
  }
}
```

### `input.messages`（必选)

仅支持单轮对话。`content` 数组:
- 文生图:`[{"text": "..."}]`
- 图像编辑:`[{"image": "..."}, ..., {"text": "..."}]`(多 image 时按数组顺序为"图1/图2/..." 对应)
- 组图生成:`[{"text": "..."}]`(不需要参考图)

### `text`

- ≤ 5000 字符,中英文标点各计 1 字符,超出截断

### `image` 输入约束

- 格式:JPEG / JPG / **PNG(不支持透明通道)** / BMP / WEBP
- 分辨率:宽高 ∈ `[240, 8000]` px,宽高比 ∈ `[1:8, 8:1]`
- 文件大小:≤ 20 MB
- **数量:0 ~ 9 张**
- 输入格式:公网 URL / Base64(`data:{MIME};base64,{data}`)

### `parameters`

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| `size` | string | `2K` | 见下方"size 规则" |
| `n` | int | `1`(非组图)/ `12`(组图) | 文生图/编辑 `1-4`;组图 `1-12` |
| `enable_sequential` | bool | `false` | 启用组图输出 |
| `bbox_list` | array | — | 交互式编辑框选,详见下方 |
| `thinking_mode` | bool | `true` | 思考模式,**仅关组图且无图片输入时生效** |
| `color_palette` | array | — | 自定义颜色主题(3-10 种 hex+ratio) |
| `watermark` | bool | `false` | "AI 生成" 水印 |
| `seed` | int | 随机 | `[0, 2147483647]` |

#### `size` 规则

**方式一(推荐):档位**

| 档位 | 总像素 | wan2.7-image-pro | wan2.7-image |
|------|--------|------------------|--------------|
| `1K` | 1024×1024 | ✓ | ✓ |
| `2K`(默认) | 2048×2048 | ✓ | ✓ |
| `4K` | 4096×4096 | **仅文生图(无图片输入、非组图)** | ✗ |

输出比例:
- 有图片输入:输出比例与输入图(多图取**最后一张**)一致,缩放到选定档位
- 无图片输入:输出**正方形**

**方式二:像素值 `宽*高`**

| 场景 | 总像素范围 | 宽高比 |
|------|------------|--------|
| wan2.7-image-pro 文生图 | `[768×768, 4096×4096]` | `[1:8, 8:1]` |
| wan2.7-image-pro 其他场景 | `[768×768, 2048×2048]` | `[1:8, 8:1]` |
| wan2.7-image 所有场景 | `[768×768, 2048×2048]` | `[1:8, 8:1]` |

> 实际输出像素可能与指定值有微小差异(对齐到 16 的倍数)。

**两种方式不可混用**。

#### `bbox_list`（交互式编辑)

`array[array[array[int]]]`,长度必须 = 输入图片数量;无需编辑的图传 `[]`。

坐标格式:`[x1, y1, x2, y2]`(左上 + 右下,绝对像素,左上角为 `(0, 0)`)。

**单张图最多 2 个 bbox**。

示例:输入 3 张图,第 1 张 2 个框,第 2 张无框,第 3 张 1 个框:
```json
[
  [[0, 0, 12, 12], [25, 25, 100, 100]],
  [],
  [[10, 10, 50, 50]]
]
```

#### `color_palette`（自定义色板)

- 3 ~ 10 种颜色(推荐 8)
- 每项 `{"hex": "#XXXXXX", "ratio": "XX.XX%"}`
- 所有 ratio 总和**必须 = 100.00%**
- **仅关组图(`enable_sequential=false`)时可用**

```json
{
  "output": {
    "choices": [
      {
        "finish_reason": "stop",
        "message": {
          "role": "assistant",

```

`content` 数组长度 = 实际生成张数(组图模式下 ≤ `n`)。

## 异步接口

### Endpoint(注意!与同步接口路径不同)

提交:
```
POST /api/v1/services/aigc/image-generation/generation
```
带 `X-DashScope-Async: enable`。

> 注意:**不是** `/multimodal-generation/generation`(同步)也不是 `/text2image/image-synthesis`(qwen 异步)。这是 wan2.7 独有路径。

轮询:
```
GET /api/v1/tasks/{task_id}
```

### 请求体结构

与同步**完全相同**(`messages` 结构、`parameters` 字段),只多了 `X-DashScope-Async: enable` 头。

### 异步响应(轮询 SUCCEEDED)

```json
{
  "request_id": "...",
  "output": {
    "task_id": "...",
    "task_status": "SUCCEEDED",
    "submit_time": "2026-03-26 17:16:01.663",
    "scheduled_time": "2026-03-26 17:16:01.716",
    "end_time": "2026-03-26 17:16:22.961",
    "finished": true,
    "choices": [
      {
        "finish_reason": "stop",
        "message": {
          "role": "assistant",
          "content": [
            {"image": "https://.../xxx.png?Expires=...", "type": "image"}
          ]
        }
      }
    ]
  },
  "usage": {
    "size": "2976*1408",
    "total_tokens": 11017,
    "image_count": 1,
    "output_tokens": 2,
    "input_tokens": 11015
  }
}
```

**与同步响应差异**:
- 异步 `output.choices` 在轮询响应里(同步也在 `output.choices`,结构对齐)
- `content` 元素多了 `"type": "image"` 字段
- `usage` 有 `total_tokens` / `input_tokens` / `output_tokens` — **均不计费**,实际按 `image_count` 张数计

## ArcReel 集成要点

### 推荐选型

ArcReel 集成 wan2.7-image / wan2.7-image-pro 作为**第二档图像后端**(与 qwen-image-2.0 并列),主要价值:
- **0~9 张参考图**(qwen-image-2.0 系列只能 1~3 张)
- **bbox 交互式编辑**(qwen 没有)
- **思考模式 `thinking_mode`** — 高质量出图
- **4K 输出**(`wan2.7-image-pro` 文生图)

### 同步还是异步?

**优先同步**:wan2.7-image 同步接口可用,响应时间通常 < 30s,没必要走异步。

仅在两种情况选异步:
- 单镜头要求 4K + thinking_mode(超 30s 风险)
- 批量入队场景(ArcReel `GenerationQueue` 模式)

ArcReel `image_backends` 当前都是同步,**保持同步实现即可**。

### Backend 实现要点

```python
async def generate(request: ImageGenerationRequest) -> ImageGenerationResult:
    content = []
    if request.reference_images:  # I2I / 多图参考
        for ref in request.reference_images[:9]:  # wan2.7 上限 9
            content.append({"image": _to_url_or_base64(ref)})
    content.append({"text": request.prompt})

    parameters = {
        "size": _normalize_size(request.size),  # "2K" 或 "1024*1024"
        "n": 1,  # ArcReel 单镜头单图
        "watermark": False,
    }
    # 文生图(无 image input)时显式开思考模式
    if not request.reference_images:
        parameters["thinking_mode"] = True

    payload = {
        "model": self.model,
        "input": {"messages": [{"role": "user", "content": content}]},
        "parameters": parameters,
    }
    # POST → output.choices[0].message.content[0].image → download
```

### `capabilities` 与 `resolutions`

```python
"wan2.7-image-pro": ModelInfo(
    capabilities=["text_to_image", "image_to_image"],
    resolutions=["1K", "2K", "4K"],  # 档位,与 gemini 风格一致
    pricing=PerImageFlat(rates={"wan2.7-image-pro": 0.5}, currency="CNY", ...),
),
"wan2.7-image": ModelInfo(
    capabilities=["text_to_image", "image_to_image"],
    resolutions=["1K", "2K"],
    pricing=PerImageFlat(rates={"wan2.7-image": 0.2}, currency="CNY", ...),
),
```

注意 `resolutions` 用档位(`1K/2K/4K`)与 gemini 一致,而 qwen-image-2.0 用像素值 — 这是不同 provider 的真实差异,backend 内自己 normalize。

### 4K 边界条件

wan2.7-image-pro 4K **仅文生图(无图片输入、非组图)**。如果 ArcReel 用户在 I2I 任务里选了 4K,backend 需要:
- 拒绝 → 抛 `ImageCapabilityError`(推荐,显式)
- 或降级到 2K + log warning

推荐拒绝,避免静默降级误导用户。

### 组图 / bbox / color_palette 暂不暴露

ArcReel 当前模型是"一镜头一图",没有组图概念,bbox 交互编辑也无对应 UI。这些参数**先不暴露到 ArcReel 前端**,backend 内部固定为关闭:

```python
parameters["enable_sequential"] = False  # 隐式默认就是 false,可省
# bbox_list / color_palette 不传
```
