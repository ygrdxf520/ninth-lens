# 千问文生图（Qwen-Image 系列)

千问图像模型族,擅长**复杂文本渲染**(多行布局、段落级文本、细粒度细节)。

通用 API 模式(Base URL、鉴权)详见 [API 概览.md](./API%20概览.md)。本文聚焦千问图像族独有的请求/响应/参数。

## 模型族结构

千问图像分两套子系:**2.0 融合系列**(生成 + 编辑同模型,同步接口)和**经典系列**(仅生成,支持同步+异步)。

### qwen-image-2.0 融合系列(推荐)

同一模型支持文生图 + 图像编辑(传 `image` 时即编辑),**仅同步接口**。

| 模型 | 简介 |
|------|------|
| `qwen-image-2.0-pro` 推荐 | Pro 版:文字渲染、真实质感、语义遵循更强(snapshot:`qwen-image-2.0-pro-2026-04-22` / `qwen-image-2.0-pro-2026-03-03`) |
| `qwen-image-2.0` 推荐 | 加速版,效果与响应速度平衡(snapshot:`qwen-image-2.0-2026-03-03`) |

**输出规格**:
- 分辨率:总像素 ∈ `[512×512, 2048×2048]`,自由设置宽高;默认 `2048*2048`
- 格式:PNG
- 张数:**1 ~ 6 张**(`n` 参数)

### 经典系列(qwen-image / plus / max,仅文生图)

| 模型 | 简介 | 同步 | 异步 |
|------|------|------|------|
| `qwen-image-max` | 真实感与自然度更强(人物质感/纹理/文字渲染最佳) | ✓ | — |
| `qwen-image-plus` | 与 `qwen-image` 能力相同,价格更优惠 | ✓ | ✓ |
| `qwen-image-plus-2026-01-09` | qwen-image-max 蒸馏加速版 | ✓ | — |
| `qwen-image` | 经典文生图 | ✓ | ✓ |

**输出规格**:
- 分辨率:**固定 5 档**,默认 `1664*928`
- 格式:PNG
- 张数:**固定 1 张**(`n=1`;传其他值会报错)

## 同步接口(推荐用法)

### Endpoint

```
POST /api/v1/services/aigc/multimodal-generation/generation
```

### 请求体(qwen-image-2.0-pro 文生图)

```json
{
  "model": "qwen-image-2.0-pro",
  "input": {
    "messages": [
      {
        "role": "user",
        "content": [
          {"text": "正向提示词..."}
        ]
      }
    ]
  },
  "parameters": {
    "negative_prompt": "低分辨率、低画质...",
    "prompt_extend": true,
    "watermark": false,
    "size": "2048*2048"
  }
}
```

### `input.messages`（必选)

**仅支持单轮对话**:数组有且只有一个元素,`role` 固定 `user`。

`content` 数组:
- 文生图:**有且只有一个** `{"text": "..."}`(传多个或不传都会报错)
- 图像编辑(2.0 系列):前 1~3 个 `{"image": "..."}` + 一个 `{"text": "..."}`

### `text`（必选,正向提示词)

- 类型:`string`
- 长度上限:
  - qwen-image-2.0 系列:**1300 token**
  - 其他模型:**800 token**
- 支持中英文,超出自动截断

### `parameters`（可选)

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| `negative_prompt` | string | — | 反向提示词,≤ 500 字符 |
| `size` | string | `2048*2048`(2.0) / `1664*928`(经典) | 详见下方"size 规则" |
| `n` | int | `1` | qwen-image-2.0 系列 `1~6`;经典系列**固定 1** |
| `prompt_extend` | bool | `true` | 智能改写正向提示词。短 prompt 提升明显,但会增加耗时 |
| `watermark` | bool | `false` | 右下角 "Qwen-Image" 水印 |
| `seed` | int | 随机 | `[0, 2147483647]`,固定提升可复现 |

#### `size` 规则

**qwen-image-2.0 系列**:格式 `宽*高`,总像素 ∈ `[512×512, 2048×2048]`。推荐档:

| size | 比例 |
|------|------|
| `2688*1536` | 16:9 |
| `1536*2688` | 9:16 |
| `2048*2048`(默认) | 1:1 |
| `2368*1728` | 4:3 |
| `1728*2368` | 3:4 |

**经典系列(qwen-image / plus / max)**:**只能从 5 档选**,默认 `1664*928`:

| size | 比例 |
|------|------|
| `1664*928`(默认) | 16:9 |
| `1472*1104` | 4:3 |
| `1328*1328` | 1:1 |
| `1104*1472` | 3:4 |
| `928*1664` | 9:16 |

### 同步响应

```json
{
  "output": {
    "choices": [
      {
        "finish_reason": "stop",
        "message": {
          "role": "assistant",
          "content": [
            {"image": "https://dashscope-result-sh.oss-cn-shanghai.aliyuncs.com/xxx.png?Expires=..."}
          ]
        }
      }
    ]
  },
  "usage": {
    "height": 2048,
    "image_count": 1,
    "width": 2048
  },
  "request_id": "..."
}
```

`output.choices[0].message.content` 是数组,每个元素含 `image` URL — n>1 时数组多元素。

## 异步接口(仅 qwen-image-plus / qwen-image)

### Endpoint

提交:
```
POST /api/v1/services/aigc/text2image/image-synthesis
```
带 `X-DashScope-Async: enable`(同视频异步规则)。

轮询:
```
GET /api/v1/tasks/{task_id}
```

### 请求体(注意与同步接口结构不同!)

```json
{
  "model": "qwen-image-plus",
  "input": {
    "prompt": "一副典雅庄重的对联..."
  },
  "parameters": {
    "negative_prompt": " ",
    "size": "1664*928",
    "n": 1,
    "prompt_extend": true,
    "watermark": false
  }
}
```

> 异步走 `input.prompt`(扁平 string),**不用 messages 结构**;长度上限 800 字符。

### 异步特有响应

`output.results` 是数组,每项含 `url` + `orig_prompt` + `actual_prompt`(智能改写后的实际 prompt — 仅异步接口返回此字段):

```json
{
  "output": {
    "task_id": "...",
    "task_status": "SUCCEEDED",
    "results": [
      {
        "url": "https://...png",
        "orig_prompt": "...",
        "actual_prompt": "..."
      }
    ]
  },
  "usage": {"image_count": 1}
}
```

## 输入图像(qwen-image-2.0 系列图像编辑模式)

`input.messages[0].content` 中插入 `{"image": "..."}`,**1 ~ 3 张**;多图时输出比例以**最后一张**为准。

`image` 支持:
- 公网 URL(http/https)
- OSS 临时 URL:`oss://dashscope-instant/...`
- Base64:`data:{MIME};base64,{data}`

**图像约束**:
- 格式:JPG / JPEG / PNG / BMP / TIFF / WEBP / GIF(GIF 只处理第一帧)
- 推荐宽高:`[384, 3072]` px(过小模糊、过大耗时)
- 文件大小:≤ 10 MB
- 输出:PNG

## ArcReel 集成要点

### 推荐选型

ArcReel 默认图像模型应选 **qwen-image-2.0-pro**(质量首选)或 **qwen-image-2.0**(速度首选)。

理由:
- 单模型覆盖 T2I + I2I(无需为编辑维护单独模型)
- 同步接口(无需异步轮询机制)
- 张数灵活(n=1~6)
- size 自由

### 不要选经典系列(qwen-image / plus / max)

- size 死板(5 档)
- 张数固定 1
- 异步路径不同(/text2image/image-synthesis vs /multimodal-generation/generation)
- 与 2.0 系列同价或更贵

### Backend 实现要点

- Endpoint:`POST {host}/api/v1/services/aigc/multimodal-generation/generation`
- `messages[0].content` 拼装规则:
  - T2I:`[{"text": prompt}]`
  - I2I:`[{"image": url1}, ..., {"image": urlN}, {"text": prompt}]`
- `parameters.size` 默认 `2048*2048`(对照 ArcReel `resolutions` 维度)
- `parameters.watermark=false` 必须显式传(默认就是 false,但显式更稳)
- `parameters.prompt_extend` 决策:
  - 默认 `true`(阿里默认)→ 模型改写 prompt,增加耗时
  - ArcReel 剧本 prompt 已是 LLM 生成的精炼描述,**建议默认 `false`** 保留原意
  - 暴露为项目级配置(类似 wan2.7-r2v)

### `capabilities` 声明(对照 ModelInfo)

```python
"qwen-image-2.0-pro": ModelInfo(
    capabilities=["text_to_image", "image_to_image"],
    resolutions=[...],  # 见下
)
```

`resolutions` 字段在 ArcReel 当前 ModelInfo 里是字符串列表 — qwen 用 `宽*高` 而非档位语义。
可选两条路:
- (a) 用阿里推荐档 `["2048*2048", "2688*1536", "1536*2688", "2368*1728", "1728*2368"]`,字面填入
- (b) 留空 `[]`,前端自由文本输入

ArcReel 当前其他 provider(gemini、ark)用的是档位关键字("1K"/"720p"),建议 (a) 保留一致体验。

### n=1~6 vs ArcReel 单镜头单图

ArcReel 当前生成每镜头出 1 张图,所以 n 始终传 1。`n` 能力不必暴露到 ModelInfo,只在 backend 内部固定即可。
