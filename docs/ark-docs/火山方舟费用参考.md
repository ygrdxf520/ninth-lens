# 视频生成模型 (Seedance) 计费说明

视频生成模型采用**异步推理**，按输出视频消耗的 token 量进行计费（元/百万 token）。

## 1. 按 Token 单价表

| **doubao-seedance-2.0**          | - 输入不含视频：46.00 - 输入包含视频：28.00        | 暂不支持                    |
| **doubao-seedance-2.0-fast**     | - 输入不含视频：37.00 - 输入包含视频：22.00        | 暂不支持                    |
| **doubao-seedance-1.5-pro**      | - 有声：16.00 - 无声：8.00                         | - 有声：8.00 - 无声：4.00   |

| **doubao-seedance-1.0-pro-fast** | 4.20                                               | 2.10                        |
| **doubao-seedance-1-0-lite**     | 10.00                                              | 5.00                        |

### 计费规则与 Token 估算说明：

- **计费条件**：仅对成功生成的视频计费。因审核等原因导致生成失败的，不收取费用。
- **视频价格估算公式**：`按 token 单价 × token 用量`
- **Seedance 2.0 / 2.0-fast token 估算**：`(输入视频时长 + 输出视频时长) × 宽 × 高 × 帧率 / 1024`
  - *注：输入包含视频时，存在最低 token 用量限制（见下方表格），若估算用量 < 最低限制，则按最低用量计费。*
- **Seedance 1.x token 估算**：`(宽 × 高 × 帧率 × 时长) / 1024`
- **Draft 视频（仅480p）估算公式**：`正常视频 token 用量 × 折算系数`
  - *注：`doubao-seedance-1.5-pro` 的 token 折算系数为：无声 0.7；有声 0.6。*
- **准确用量**：以调用 API 后返回信息中的 `usage` 字段为准。

图片生成模型按**成功输出的图片数量**（元/张）进行计费。

### 计费规则说明：

- **计费条件**：按成功输出的图片数量计费。因审核等原因未成功输出的图片不计费。
- **组图场景**：按实际生成的图片数量计费。

# 大语言模型 (文本生成) 计费说明

大语言模型按 token 后付费，费用公式：`输入单价 × 输入 token + 输出单价 × 输出 token`。

部分模型按输入长度分段计费，不同区间 token 单价不同。
