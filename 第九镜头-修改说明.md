## 🚀 如何使用

### 方式一：Web 端部署（推荐服务器使用）

#### Docker 部署
```bash
cd deploy
cp .env.example .env
# 修改 .env 中的配置，建议设置 AUTH_ENABLED=false
docker compose up -d
# 访问 http://localhost:1241
```

#### 本地开发运行
```bash
# 前端
cd frontend
pnpm install
pnpm dev

# 后端（另一个终端）
uv run uvicorn server.app:app --reload --port 1241
```

### 方式二：桌面端（推荐本地使用）

#### 快速开始（开发模式）
```bash
# 1. 先启动后端
cd ArcReel
uv run uvicorn server.app:app --port 1241

# 2. 再启动 Electron（另一个终端）
cd desktop
npm install
npm start
```

#### 打包成 EXE 安装包（Windows）

**一键打包（推荐）：**
```bash
cd desktop
build-windows.bat
```

**手动打包：**
详细步骤请参考 `desktop/桌面端打包说明.md`

## ⚠️ 注意事项

### 1. 后端鉴权
前端跳过了登录页面，但后端 API 可能仍需要 token 验证。

**解决方案：**
- 设置环境变量 `AUTH_ENABLED=false` 完全禁用后端鉴权
- 桌面端已默认设置此变量

### 2. 文档文件
README、CHANGELOG 等文档文件仍保留 ArcReel 原始内容，如需修改请自行调整。

### 3. 许可证
本项目基于 ArcReel (AGPL-3.0) 修改，仍需遵守 AGPL-3.0 许可证。
- 可以商用
- 但修改后的代码需要开源
- 做 SaaS 服务也需要开源
- 如需闭源商用，请联系原作者购买商业授权

### 4. 桌面端打包
- 必须在 **Windows 环境**下打包，才能生成 Windows EXE
- 需要安装 Python 3.12+、Node.js 20+、pnpm
- 打包后的安装包约 150-200MB
- PyInstaller 打包可能会遇到缺少模块的问题，需要在 `backend.spec` 中添加隐藏导入

### 5. AI 供应商配置
桌面端安装后，首次使用需要在系统设置中配置 AI 供应商的 API Key：
- 火山方舟
- OpenAI
- Google Gemini
- Grok
- 等等
